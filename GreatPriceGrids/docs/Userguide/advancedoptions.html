Advanced Options

Replaceing locked items with custom text

Author

Tim Colley


Advanced options are locked by default. This is because they are items that appear on the price grid that don't normally change. Some of these items might well cause problems later on in the production process if they are changed. For this reason, should these controls be activated, a system log entry will be created, detailing that the controls were enabled, for which tour and the current date and time.

That being said, there are times when you may really need to change these, a tour code being changed half way though brochure production for example, this is what each of the controls in the (now unlocked) advanced options section are for.

  • Override the tour code and/or tour title
  • Override the deposit and/or single supplement
  • Change the duration of the tour or the class of rail that is used. Note: if you change tha class of rail, and the original rail class is split (eg. First/Standard, First & Standard/Standard), then this override will replace all of the rail classes with the singular rail class that you have chosen. If you need to do this on a multi-class grid, please amend the grid after it has been produced to avoid the wrong type of grid being built.
