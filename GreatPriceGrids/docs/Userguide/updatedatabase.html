Update Database

What to do to get the latest prices

Author

Tim Colley


To provide you with the most accurate price grids, GPG must be regulaly updated to the lates tour definitions. These are in files called CSVs. Basically they are are spreadsheets produced by IT from Traveller. You can view these in excell if you wish, but GPG is designed to give you a nice, easy to use inteface.

when you click the 'Update Database' button, you will be sown a File Cooser window. To update your database, please do the following:

  • Navigate to 'Marketing Operations > Marketing > Prices'
  • Select the flavour of the prices that you need (GRJ, GCJ or RD. as of 21/06/18 only GRJ is supported so for now ignore the other two)
  • Double click the file, or select the file and click 'Open'
  • Wait for the GPG window to update (this may take a few seconds)

When this process has finished, you will have the latest prices form Traveller imported into your app.

HELPFUL TIP: The system that updates these definition files only runs twice a day. To acheve the best results, update your database at around 10am or at around 4pm. This will ensure that enough time has passed for the latest version to be placed into the prices folder.
