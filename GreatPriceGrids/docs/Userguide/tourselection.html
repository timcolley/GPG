Tour Selection

Selecting your required tour or searching for it.

Author

Tim Colley


After you have updated your database, you can go on to building your price grid. Here there are several things to consider, but lets start with the fist think in the list: finding the correct tour.
  • You will see in the main window, a section labelled as 'Filters & Options', thes is the part of the interface that we will use to find the tour we need.

  • Click the button labelled 'Tour Code', the button will expand to show you a list of all available tour codes.

  • Select the tour code of the tour you need a price grid for. The whole window will then update, displaying all information for that tour. Additionally, you will see a list of available departures in the preview panel to the right hand side.
TIP: If you want to search (the list of tours can be quite long) then click the button, type the three character tour code on your keyboard, then the return key. This will take you directly to the tour you require. If the tour doesn't exist (e.g. it's not live on Traveller or your pagination lists an incorrect tour code), then the closest match will be automatically selected for you.
