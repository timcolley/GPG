Standard Options

Setting up your grid with optional extras.

Author

Tim Colley


Standerd options are always unloct ans usable. These are unrestricted and can be changed at any time.

In the standard options, you can override (change the behaviour) of several elements of the finished price grid. You can do the following with these options:

  • Change the departure location, if listed incorrectly


  • Change the visa requirements, again, if incorrect


  • Add economy class flights to the rail class box on the price grids


  • Add a Fly:Home option box to the bottom of a price grid


  • Force the price grid to include regional departures, if these are missing
  • Add or remove information for the regional departures and their prices


  • Update the normal small print text, so it's more applicable to your brochure. E.g. adding the page numbers for the terms and conditions
  • Add a 'Late Departures' box to the top of the grid and/or remove the tour title from the grid, these are normally used when building single pages or single page tactical brochures.
You may not need to change any of these settings, but these controls are there to make it easier for you to produce correct price grids, first time, even in the event of the original Traveller data being inaccurate.
